# CHANGELOG Generator

A bash script that automatically generates a structured `CHANGELOG.md` from git history.

## Quick Start (3 Steps)

```bash
# 1. Clone or download this script
git clone https://github.com/universe7creator/claude-builders-bounty.git
cd claude-builders-bounty

# 2. Run the script on your repo
bash changelog.sh /path/to/your/repo

# 3. Check generated CHANGELOG.md
cat CHANGELOG.md
```

## Features

- ✅ Fetches commits since the last git tag
- ✅ Auto-categorizes into: Added / Fixed / Changed / Removed
- ✅ Outputs properly formatted CHANGELOG.md
- ✅ Works with any GitHub repository
- ✅ Zero dependencies (pure bash)

## Usage

```bash
# Basic usage (uses current directory)
bash changelog.sh

# Specify repository path
bash changelog.sh /path/to/repo

# Specify output file
bash changelog.sh /path/to/repo CUSTOM_CHANGELOG.md
```

## Requirements

- Bash 4.0+
- Git
- Standard Unix tools (grep, sed, date)

## How It Works

1. Finds the last git tag in the repository
2. Fetches all commits since that tag
3. Categorizes commits based on keywords:
   - **Added**: add, new, feature, implement, create
   - **Fixed**: fix, bug, resolve, repair, hotfix
   - **Changed**: update, change, modify, refactor, improve
   - **Removed**: remove, delete, deprecate, drop
4. Outputs formatted CHANGELOG.md
